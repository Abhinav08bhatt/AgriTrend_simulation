# Source Code

> This folder contains all core source code for AgriTrend_simulation.

## Modules Overview

- `data_generation.py` – Creates synthetic agricultural data
- `data_loader.py` – Saves and loads datasets
- `analysis.py` – Runs regression and factor analysis
- `model.py` – Trains yield prediction models and forecasts futures
- `scenarios.py` – Applies intervention and improvement scenarios
- `visualization.py` – Generates graphs and plots
- `report_builder.py` – Builds the final PDF report
- `paths.py` – Centralized path management
