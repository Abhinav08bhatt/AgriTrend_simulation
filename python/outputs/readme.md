# Outputs

This folder stores all generated outputs produced by the system.

It includes graphs and PDF reports created during analysis and simulation runs.
These files are generated automatically and do not require manual editing.

- ### [graphs](../outputs/graphs)
    > This folder contains all visualizations generated by the simulation.

    - These graphs illustrate historical trends, factor relationships, future projections, and scenario comparisons.
    - Graphs are generated automatically and saved for use in reports and documentation.

- ### [reports](../outputs/reports)
    > This folder contains generated PDF reports.

    - The main report summarizes the full simulation, analysis, and scenario results in a readable format.
    - Reports are generated automatically using the report builder module.
