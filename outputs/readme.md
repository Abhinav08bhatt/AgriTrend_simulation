# Outputs

This folder contains all files generated by the AgriTrend_simulation system during
analysis and scenario runs.

All contents here are **automatically produced** by the codebase.  
These files are intended for inspection, reporting, and comparison, and should
not be edited manually.

## Folder Structure

- `graphs/`
  Contains all visualizations generated during data analysis, modeling, and
  scenario comparison.

- `reports/`  
  Contains automatically generated PDF reports that summarize the selected
  dataset, scenario, and results in a structured, readable format.

## Notes

- Output files are overwritten or regenerated on each run
- Filenames may vary depending on the selected dataset and scenario
- The outputs folder can be safely cleared and regenerated at any time
