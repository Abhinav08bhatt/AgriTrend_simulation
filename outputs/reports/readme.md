# Reports

This folder contains automatically generated PDF reports produced by the
AgriTrend_simulation system.

Each report summarizes:
- the selected dataset
- the chosen scenario
- historical trends
- baseline future projections
- scenario comparisons
- conclusions and limitations

## Report Characteristics

- Reports are generated programmatically using the report builder module
- Filenames include dataset and scenario information for clarity
- Content is designed to be scenario-aware and interpretation-safe

## Usage Notes

- Reports are not meant to be edited manually
- Regenerating a report will overwrite files with the same name
